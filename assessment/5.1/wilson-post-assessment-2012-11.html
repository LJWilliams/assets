Sample Pre-Assessment Questions

by Greg Wilson

Q1

A user opens a new shell window and types:

cat humanoids.txt

The output is:

81 troll
2  goblin
5  orc

She then types:

sort humanoids.txt

The output is:

2  goblin
5  orc
81 troll

because:

  1. The number 2 is less than the number 5, and the number 5 is less than the number 81.
  2. The letter 'g' comes before the letter 'o', and the letter 'o' comes before the letter 't'.
  3. The output would actually be different than what's shown.
  4. Other: ____________________

Q2

A user opens a new shell window and types:

for filename in goblin.txt orc.txt troll.txt
do
    echo 'file:' $filename
    cat $filename
    echo
done

The output is:

file: goblin.txt
goblin

file: orc.txt
orc
orc

file: troll.txt
troll
troll
troll

She then creates a shell script called contents.sh in the same directory and puts the following commands in it:

for species in *
do
    echo 'file:' $species
    cat $species
    echo
done

Does this produce the same output as the original interactive command?

  1. Yes, but only if there are no other files in the same directory.
  2. Yes, but only if she runs the script using bash contents.sh *.txt.
  3. No, because species isn't a file name.
  4. Other: ____________________

Q3

The file humanoids.txt contains:

81 troll
2  goblin
5  orc

A user writes the following Python program:

values = []
for line in 'humanoids.txt':
    fields = line.split()
    first = fields[0]
    values.append(first)
print values[0]

What is the output?

  1. The number 8, because that is the first digit of 81 + 2 + 5.
  2. The string '81 troll', because that is the first line of humanoids.txt.
  3. The letter 'h', because that is the first character of humanoids.txt.
  4. Other: ____________________

Q4

A user writes the following Python program:

total = 0
numbers = [1, 2, 3]
for outer in numbers:
    for inner in ______:
        total += outer
print total

What should the user put in the blank so that the program's output is 14?

  1. numbers[0:total]
  2. numbers[0:outer]
  3. numbers.reverse()
  4. Other: ____________________
